Introduction to HTML

What is Internet ?

The Internet is a global system of interconnected computer networks that use the Internet protocol suite (TCP/IP) to communicate between networks and devices. It's essentially a vast network connecting millions of computers worldwide, allowing for the sharing of information and communication between users.

What is browser ?

A browser is an application software used to locate, retrieve and display content on the World Wide Web, including webpages, images, videos and other types of files. Most browsers can be used to access information over the internet as well as local networks or files stored on your computer.

What is HTTP?

An HTTP server is a software application or hardware that accepts requests from clients via the Hypertext Transfer Protocol (HTTP) and responds with web content. It's the core component that allows users to access and view websites. Essentially, it stores, processes, and delivers web pages to clients, such as web browsers.

What is HTTPS ?

An HTTPS server is a web server that uses the HTTPS protocol to securely transmit data between a web browser and a website. It encrypts the communication, ensuring that sensitive information like passwords and credit card details are protected from being intercepted.

What is website ?

A website is a collection of webpages that are accessed with a single domain name or URL. It includes all the individual pages, multimedia details and files stored on a web server. Websites are built using languages like HTML, CSS and JavaScript to make visually appealing layouts and functionality.

What is Web app ?

A web application, or web app, is a software program that runs on a remote server and is accessed through a web browser over the internet.
